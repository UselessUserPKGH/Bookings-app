# Contact Book — Контактная книга

Бельченко, Долгов<br>
Простое Android‑приложение «Контактная книга» с одним экраном, реализующее базовые действия с контактными данными с помощью `Intent` и корректно настроенной `exported`‑разметкой компонентов.

## Функционал

Приложение предоставляет на главном экране четыре кнопки:

1. **«Позвонить»**  
   Открывает системный набор номера с предзаполненным номером `+7 (495) 123‑45‑67`.

2. **«Написать email»**  
   Открывает почтовое приложение с предзаполненным адресом `contact@example.com` и темой письма «Обращение».

3. **«Показать офис на карте»**  
   Открывает карты с координатами `60.0237, 30.2289` и меткой «Наш офис».

4. **«Поделиться контактом»**  
   Открывает диалог выбора приложения (`Intent.createChooser`) для отправки текста вида:  
   `Контакт: +7 (495) 123‑45‑67, contact@example.com`.

## Обработка отсутствующих приложений

Для всех четырёх действий реализована проверка, что подходящее приложение существует в системе.  
Если приложение‑обработчик не найдено, пользователь видит сообщение `Toast` с уведомлением о неподходящем приложении.

## Особенности реализации

- Все активити и другие компоненты, которые ожидают внешних вызовов, помечены атрибутом `android:exported` в соответствии с требованиями Android‑манифеста.
- Для кнопки «Поделиться» используется `Intent.createChooser`, чтобы всегда показывать диалог выбора приложения‑получателя, даже если у пользователя есть настройка по умолчанию.
- Интерфейс выполнен в одном экране, с ёмкими подписями и понятной иконографикой, ориентированной на удобство и эстетику.

## Требования к сборке

- Android Studio 2023+  
- Минимальный уровень API: 21  
- Зависимости: стандартный Android SDK (без внешних библиотек).  

## Как использовать

1. Откройте проект в Android Studio.  
2. Запустите на устройстве или эмуляторе.  
3. Нажмите нужные кнопки для звонка, email, карты или шаринга контакта.
